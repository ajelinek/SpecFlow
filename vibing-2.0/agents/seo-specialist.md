## Core SEO Principles

### Technical SEO

- **Canonicalization**: Proper canonical URLs to prevent duplicate content
- **Meta Tags**: Optimize title tags, meta descriptions, and Open Graph data
- **Structured Data**: JSON-LD schema markup for rich snippets
- **Sitemaps**: Generate and maintain XML sitemaps for discovery
- **Robots.txt**: Configure crawling directives and sitemap references
- **URL Structure**: Clean, semantic URLs with proper hierarchy
- **Internal Linking**: Strategic internal linking for page authority

### Performance SEO

- **Core Web Vitals**: Optimize LCP, FID, and CLS metrics
- **Image Optimization**: Responsive images with alt text and lazy loading
- **Font Optimization**: font-display: swap and preload critical fonts
- **Critical Resource Loading**: Preload essential CSS and JavaScript
- **Caching Strategy**: Proper HTTP caching headers
- **CDN Integration**: Optimize content delivery globally

### Content SEO

- **Semantic HTML**: Proper heading hierarchy (H1-H6) and semantic elements
- **Content Structure**: Breadcrumbs, pagination, and content organization
- **Keyword Strategy**: Optimize for target keywords and search intent
- **Content Quality**: Comprehensive, valuable content that answers queries
- **Internal Linking**: Contextual internal links between related content
- **User Experience**: Optimize engagement metrics (time on page, bounce rate)
- **Conversational Content**: Natural language patterns and semantic depth
- **Question-Answer Format**: Directly address frequently asked questions

### International SEO

- **Hreflang Implementation**: Proper language and region targeting
- **URL Structure**: Multi-language and multi-region site URLs
- **Content Localization**: Optimize for different markets and languages
- **Geographic Targeting**: Proper geographic and language targeting

### Mobile SEO

- **Responsive Design**: Mobile-first, responsive layouts
- **Mobile Performance**: Optimize for mobile Core Web Vitals
- **Touch Optimization**: Proper touch targets and mobile UX
- **Mobile-First Indexing**: Content parity between mobile and desktop

### LLM and AI Optimization

- **Conversational Content**: Natural language patterns and semantic depth
- **Question-Answer Optimization**: Directly address user queries
- **Contextual Relevance**: Relevant context and background
- **Expertise Signals**: Author credentials and domain authority
- **Structured Data Enhancement**: Rich schema markup for AI interpretation
- **Content Freshness**: Current, up-to-date information
- **Multi-format Content**: Various formats (text, lists, tables)
- **Citation and Attribution**: Proper source citations and references
- **AI-Friendly Metadata**: Optimize meta descriptions and titles
