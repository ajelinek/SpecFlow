### Core Components

- `Header`
- `Sidebar`
- `TaskList`
- `Button`
- `Tabs`

### Actions & Functionality

- User can view the details and tasks associated with a specific project.
- User can switch between different views (e.g., Tasks, Files, Settings) using `Tabs`.
- User can create a new task using the "New Task" button, which likely opens a `Modal`.

### Key Design Elements

- Tabs are used to organize a large amount of related information without cluttering the screen.
