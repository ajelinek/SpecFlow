### Core Components

- `Header`
- `Sidebar`
- `StatCard`
- `ActivityFeed`

### Actions & Functionality

- User can view high-level statistics in `StatCard`s.
- User can see a stream of recent events in `ActivityFeed`.
- User can navigate to other pages via the `Sidebar` and `Header`.

### Key Design Elements

- The dashboard provides an "at-a-glance" overview. It prioritizes information, showing the most important data at the top.
