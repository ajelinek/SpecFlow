### Core Components

- `Header`
- `Sidebar`
- `ProfileForm`
- `Tabs`

### Actions & Functionality

- User can update their personal information.
- User can manage account-level settings.
- User can save changes, with feedback on success or failure.

### Key Design Elements

- Settings are grouped into logical sections using `Tabs` for clarity.
- A single "Save Changes" button per tab provides a clear action for the user.
