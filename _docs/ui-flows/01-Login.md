```
|  Password: [__________________]          |

|            +-----------------+           |
|                                          |
|  Forgot Password? | Create Account      |

```

### Core Components

- `LoginForm`
- `Button`
- `Input`
- `AuthLayout`

### Actions & Functionality

- User can enter their email and password.
- User can click "Login" to authenticate.
- System displays validation errors for incorrect inputs.
- System displays a global error for failed login attempts.

### Key Design Elements

- The layout is centered and simple to focus the user on the single task of logging in.
